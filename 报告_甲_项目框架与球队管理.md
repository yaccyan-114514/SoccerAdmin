# 项目1：中超联赛赛事管理系统

## 1 项目说明

### 项目功能

本项目是一个中超联赛赛事管理系统，实现了以下核心功能：
- **球队管理**：管理中超16支球队的基本信息（名称、城市、主场、队徽等）
- **球员管理**：管理各球队球员信息及统计数据（进球、助攻、红黄牌、身价等）
- **新闻展示**：发布和展示中超联赛相关新闻资讯
- **联赛积分榜**：展示各赛季球队积分排名与多维统计数据
- **赛程查询**：查询历年中超联赛赛程安排
- **在线购票**：用户登录后可选座购票，集成支付宝沙箱支付

### 所用技术

| 类别 | 技术 |
|------|------|
| 后端框架 | Python 3 + Django 6.0.2 |
| 数据库 | SQLite3 |
| 前端 | HTML5 + CSS3 + JavaScript + Bootstrap |
| 支付集成 | 支付宝沙箱环境（python-alipay-sdk） |
| 版本控制 | Git |

### 运行平台

- 操作系统：macOS / Windows / Linux
- Python版本：3.10+
- 浏览器访问：http://127.0.0.1:8000/

# core/views.py

def home(request):
    """首页：顶部按钮、新闻、积分榜"""
    news_list = News.objects.all()[:5]
    standings = TeamStanding.objects.select_related("team").filter(
        year=2026
    ).order_by("-points", "-goals_for", "goals_against")
    context = {
        "news_list": news_list,
        "standings": standings,
    }
    return render(request, "core/home.html", context)

class TeamAdmin(admin.ModelAdmin):
    list_display = ["short_name", "name", "city", "stadium", "founded_year"]

    search_fields = ["name", "short_name", "city"]

class PlayerAdmin(admin.ModelAdmin):
    list_display = ["name", "team", "year", "position", "number",
